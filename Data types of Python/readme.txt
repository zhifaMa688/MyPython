数据类型是Python学习的基础。

Python 的数据类型主要包括以下几个：
整型int：整数，例如2、7；
浮点型float：小数，例如1.3、2.8；
字符串string：一串有序的字符，例如“hello”；
布尔型bool：逻辑运算类型，True或者false；
列表list：多类型多个元素的有序序列，例如[100,2.5,"hello",True]；
元组tuple：多类型多元素有序且不可变序列，例如(100,2.5,"hello",True)；
字典dictionary：无序的键值对，{"键1":"值1","键2":"值2","键3":"值3"...};		//不能出现重复键值
集合set：无序且无重复元素的序列：{"A","B","C","D"};	//没有值的字典？
